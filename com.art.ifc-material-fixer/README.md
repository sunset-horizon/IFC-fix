# ART IFC Material Fixer

Editor-only Unity package for restoring IFC model colors from OBJ/MTL files generated by IfcConvert.

## Requirement

- IFC-Unity-Editor-Plugin must be installed in the project.
- Its editor assembly must be named `IfcUnityEditorPlugin`.

## Menus

- `IFC Tools/Load IFC file with OBJ MTL materials`
  - Recommended entry.
  - Runs the existing IFC loader, then automatically applies colors from the generated OBJ/MTL files.

- `Tools/IFC/Load IFC File With OBJ MTL Materials`
  - Same as above, under the ART tool menu.

- `Tools/IFC/Fix Selected IFC Materials Auto Find OBJ MTL`
  - Uses the selected IFC root object name to find matching OBJ/MTL files in the IFC plugin output folder.

- `Tools/IFC/Fix Selected IFC Materials From OBJ MTL`
  - Lets you manually select OBJ and MTL files for the selected IFC root object.

## Notes

- The tool maps Unity renderers to OBJ `o`/`g` names and `usemtl` entries.
- Material colors are read from MTL `Kd` diffuse color values.
- Materials are created under `Assets/Materials/IfcMtlColors`.
- The preferred shader is `Universal Render Pipeline/Lit`; the fallback shader is `Standard`.
